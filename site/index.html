Bridge for PrusaLink

A browser extension that lets web apps send G-code to your own PrusaLink printers over the LAN, gated by a per-origin consent prompt. Printer URLs and credentials stay in the extension; a page only ever sees a name, model, and status.

GitHub · npm · client docs

Unofficial; not affiliated with Prusa Research. Not on the stores yet — load a build from Releases unpacked, or build from source.
